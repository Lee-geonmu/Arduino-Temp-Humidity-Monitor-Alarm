# 온습도 모니터링 및 알람 시스템

## 프로젝트 소개
DHT11 센서로 온습도를 측정하여 LCD에 실시간 표시하고, 
설정한 임계값(온도 30도, 습도 70%) 초과 시 LED와 부저로 경고하는 아두이노 프로젝트

## 사용 부품
- Arduino Uno
- DHT11 온습도 센서
- 16x2 LCD (핀: 12, 11, 5, 4, 3, 2)
- LED (핀 13)
- 부저 (핀 8)

## 주요 기능
- 이동평균 필터(5개 샘플)로 센서 노이즈 제거
- 부팅 시 캘리브레이션 단계 수행
- 디바운싱 로직: 3회 연속 임계값 초과 시에만 알람 발동
- 센서 읽기 실패 시 에러 메시지 표시
